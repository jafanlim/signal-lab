# Command: /run-migration

## What it does
Runs Prisma migration safely and validates the result.

## Steps it performs
1. `npx prisma format` — formats schema.prisma
2. `npx prisma migrate dev --name <auto-generated from schema diff>` — creates and runs migration
3. `npx prisma generate` — regenerates the client
4. `npx prisma validate` — confirms schema is valid
5. Reports any errors with remediation steps

## When to use
After any change to backend/prisma/schema.prisma
